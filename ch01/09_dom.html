① DOM とは

重要ポイント：DOM（Document Object Model）はHTMLをJavaScriptから操作するための インターフェースです。HTMLの各要素がノード（節）としてツリー構造で管理されます。

DOM ツリーの構造

📄 document
<html>
<head> ← メタ情報
<body>
<h1 id="title">
"Hello, World!" ← テキストノード
<p class="desc">
"説明文..."
<ul id="list">
<li> <li> <li> 
// DOM の基本的な考え方
// HTML → DOM ツリー → JavaScript で操作

// document オブジェクトがエントリーポイント
console.log(document.title);    // → ページのタイトル
console.log(document.URL);      // → 現在のURL
console.log(document.body);     // → body 要素
console.log(document.head);     // → head 要素

// ノードの種類
// Element ノード：<div>, <p> など HTML 要素
// Text ノード：要素内のテキスト
// Attribute ノード：id, class などの属性

② 要素の取得

重要ポイント：getElementById・querySelector・ querySelectorAll などで DOM 要素を取得します。 現代のJavaScriptでは querySelector 系が推奨されます。

要素取得メソッド一覧

メソッド取得対象戻り値推奨度
getElementById(id) id 属性で1つ取得 Element / null ⭐⭐⭐
querySelector(css) CSSセレクタで最初の1つ Element / null ⭐⭐⭐⭐（推奨）
querySelectorAll(css) CSSセレクタで全て NodeList ⭐⭐⭐⭐（推奨）
getElementsByClassName(cls) class 名で全て HTMLCollection（ライブ） ⭐⭐
getElementsByTagName(tag) タグ名で全て HTMLCollection（ライブ） ⭐⭐

// getElementById：id で1つ取得
const title = document.getElementById("demo-title");
console.log(title.textContent); // → "タイトル要素..."

// querySelector：CSSセレクタで最初の1つ
const firstText = document.querySelector(".demo-text");
const btn       = document.querySelector("#demo-btn");
const firstLi   = document.querySelector("li:first-child");

// querySelectorAll：CSSセレクタで全て（NodeList）
const allTexts = document.querySelectorAll(".demo-text");
console.log(allTexts.length); // → 2

// NodeList は forEach で反復できる
allTexts.forEach(el => console.log(el.textContent));

// data 属性でのセレクタ
const item1 = document.querySelector('[data-index="1"]');

③ テキスト・属性・スタイルの変更

重要ポイント：取得した要素の textContent・innerHTML・ setAttribute・style・classList を使って 要素の内容・属性・見た目を変更できます。

🎮 ライブ DOM 操作デモ

textContent / innerHTML

← テキストを入力して変更してみましょう
const el = document.getElementById("live-target");

// textContent：テキストのみ（HTMLタグはエスケープ）
el.textContent = "Hello, DOM!";

// innerHTML：HTMLタグも解釈される（XSS に注意！）
el.innerHTML = "<strong>太字</strong>テキスト";

// innerText：レンダリング後のテキスト（CSS を考慮）
el.innerText = "表示テキスト";
⚠️ innerHTML はXSSに注意！
ユーザーが入力した値をそのまま innerHTML に設定すると スクリプトインジェクション攻撃（XSS）の危険があります。 ユーザー入力には必ず textContent を使いましょう。

style プロパティ

← プロパティと値を選んで変更してみましょう
const el = document.getElementById("live-target");

// style プロパティ（インラインスタイル）
el.style.backgroundColor = "#ff6b6b";
el.style.color            = "#fff";
el.style.fontSize         = "24px";
el.style.borderRadius     = "12px";

// CSS プロパティ名はキャメルケースで指定
// background-color → backgroundColor
// border-radius    → borderRadius

// 複数のスタイルをまとめて変更
el.style.cssText = "background:#ff6b6b; color:#fff; padding:20px;";

classList API

← クラス操作を選んで実行してみましょう
const el = document.getElementById("live-target");

// classList API
el.classList.add("highlight");         // クラスを追加
el.classList.remove("highlight");      // クラスを削除
el.classList.toggle("highlight");      // あれば削除、なければ追加
el.classList.contains("highlight");   // → true / false
el.classList.replace("old", "new");  // クラスを置換

// 複数クラスを一度に追加
el.classList.add("active", "visible", "animated");

// className で全クラスを確認
console.log(el.className); // → "highlight active visible animated"

属性の操作

← 属性操作を選んで実行してみましょう
const el = document.getElementById("live-target");

// 属性の取得・設定・削除・確認
el.getAttribute("id");              // → "live-target"
el.setAttribute("title", "ヒント"); // title 属性を設定
el.removeAttribute("title");       // title 属性を削除
el.hasAttribute("id");             // → true

// data-* 属性（dataset）
// <div data-user-id="123" data-role="admin">
el.dataset.userId; // → "123"（data-user-id → userId）
el.dataset.role;   // → "admin"
el.dataset.newKey = "value"; // data-new-key を追加

④ 要素の追加・削除

重要ポイント：createElement で新しい要素を作成し、 appendChild や insertBefore で DOM に追加します。 remove() や removeChild() で削除できます。

🎮 ライブ リスト操作デモ

↓ 下のコントロールでリストを操作してみよう

  • 初期アイテム 1
  • 初期アイテム 2
  • 初期アイテム 3
← リストを操作してみましょう 
// 要素の作成と追加
const ul = document.getElementById("live-list");

// 1. createElement で新しい要素を作成
const li = document.createElement("li");
li.textContent = "新しいアイテム";

// 2. DOM に追加
ul.appendChild(li);          // 末尾に追加
ul.prepend(li);              // 先頭に追加
ul.insertBefore(li, ul.firstChild); // 指定位置の前に挿入

// insertAdjacentHTML（位置を指定して HTML を挿入）
ul.insertAdjacentHTML("beforeend", "<li>末尾に追加</li>");
ul.insertAdjacentHTML("afterbegin", "<li>先頭に追加</li>");

// 要素の削除
const lastLi = ul.lastElementChild;
lastLi.remove();              // 要素自身を削除
ul.removeChild(lastLi);      // 親から子を削除

// 全子要素を削除
ul.innerHTML = "";           // innerHTML を空にする
while (ul.firstChild) {
  ul.removeChild(ul.firstChild);
}

要素のナビゲーション（親・子・兄弟）

const el = document.querySelector("#live-list li");

// 親・子・兄弟要素へのアクセス
el.parentElement;          // 親要素（ul）
el.parentNode;             // 親ノード（テキストノードも含む）
el.children;               // 子要素のコレクション
el.firstElementChild;      // 最初の子要素
el.lastElementChild;       // 最後の子要素
el.nextElementSibling;     // 次の兄弟要素
el.previousElementSibling; // 前の兄弟要素

// closest：祖先要素を CSS セレクタで検索
el.closest("ul");           // 最も近い ul 祖先要素
el.closest(".container");  // 最も近い .container 祖先要素

⑤ DOMContentLoaded と script の配置

重要ポイント：JavaScriptが実行されるタイミングによっては、 まだ DOM が構築されていない場合があります。 DOMContentLoaded イベントや defer 属性を使って DOM の準備完了後に実行するようにします。

DOMContentLoaded イベント

// DOMContentLoaded：HTML の解析完了後に実行
// （画像・CSS の読み込み完了は待たない）
document.addEventListener("DOMContentLoaded", () => {
  // ここで DOM 操作を行う
  const title = document.getElementById("title");
  title.textContent = "DOM 準備完了！";
});

// load：すべてのリソース（画像・CSS）読み込み完了後
window.addEventListener("load", () => {
  console.log("すべて読み込み完了");
});

// readyState で現在の状態を確認
console.log(document.readyState);
// → "loading"（解析中）
// → "interactive"（DOMContentLoaded 後）
// → "complete"（load 後）

defer / async 属性

<!-- 通常：HTML の解析をブロックする -->
<script src="app.js"></script>

<!-- defer：HTML 解析後に実行（順序保証）-->
<script src="app.js" defer></script>

<!-- async：ダウンロード完了次第実行（順序不保証）-->
<script src="app.js" async></script>
属性ダウンロード実行タイミング順序推奨場面
なし HTML をブロック 即時 保証 インラインスクリプト
defer 並行 DOM 構築後 ✅ 保証 DOM 操作が必要なスクリプト
async 並行 ダウンロード完了後 ❌ 不保証 独立した解析スクリプト

script タグの配置場所

<!-- ❌ head 内（DOM が未構築で要素を取得できない）-->
<head>
  <script>
    document.getElementById("title"); // → null（まだ存在しない）
  </script>
</head>

<!-- ✅ body の末尾（DOM が構築済み）-->
<body>
  <h1 id="title">タイトル</h1>
  <!-- ... -->
  <script src="app.js"></script> <!-- ← ここに配置 -->
</body>

<!-- ✅ defer 属性（head 内でも安全）-->
<head>
  <script src="app.js" defer></script>
</head>
