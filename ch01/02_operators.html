① 値の加算

重要ポイント：+ 演算子は数値の加算と文字列の連結の両方に使われます。
オペランド（演算対象）の型によって動作が変わるため注意が必要です。

🧮 インタラクティブ電卓（加算）

2つの数値を入力して加算を試してみましょう
+
← 数値を入力して計算してみましょう
// 数値の加算
let a = 10;
let b = 5;
let sum = a + b;
console.log(sum); // → 15

// 変数を使わずに直接計算
console.log(100 + 200); // → 300
console.log(3.14 + 1.5); // → 4.640000000000001（浮動小数点の誤差）

// 数値と文字列の + は文字列連結になる（注意！）
console.log(10 + "5");  // → "105"（文字列）
console.log("10" + 5);  // → "105"（文字列）
⚠️ 浮動小数点の誤差に注意
0.1 + 0.2 は 0.30000000000000004 になります。 これはJavaScriptに限らず、IEEE 754 浮動小数点数を使うすべての言語で起きる現象です。 金額計算などでは整数に変換してから計算するか、専用ライブラリを使いましょう。

重要ポイント：JavaScriptには加算以外にも減算・乗算・除算・剰余・べき乗の演算子があります。

let a = 10;

console.log(a - b);  // → 7    （減算）

// 剰余（%）の活用例：偶数・奇数の判定
console.log(7 % 2); // → 1（奇数）

// Math オブジェクトも活用できる
console.log(Math.floor(10 / 3)); // → 3  （切り捨て）
console.log(Math.round(3.7));      // → 4  （四捨五入）
console.log(Math.abs(-5));         // → 5  （絶対値）
console.log(Math.sqrt(16));        // → 4  （平方根）

インクリメント・デクリメントの前置と後置の違い（重要！）

前置（++x）と後置（x++）の違いを確認しよう
← 種類と初期値を選んで実行してみましょう
// 短縮記法の基本
let x = 10;
x += 5;   console.log(x); // → 15
x -= 3;   console.log(x); // → 12
x *= 2;   console.log(x); // → 24
x /= 4;   console.log(x); // → 6
x %= 4;   console.log(x); // → 2

// 前置インクリメント：先に +1 してから値を返す
let a = 5;
console.log(++a); // → 6（+1した値を返す）
console.log(a);   // → 6

// 後置インクリメント：現在の値を返してから +1 する
let b = 5;
console.log(b++); // → 5（元の値を返す）
console.log(b);   // → 6（その後 +1 される）

④ 文字列と連結

重要ポイント：文字列は + で連結できます。また ES6 以降は`テンプレートリテラル`を使うとより簡潔に書けます。

文字列連結のビジュアル

"Hello"
+
" World"
=
"Hello World"

インタラクティブ：文字列を連結してみよう

2つの文字列を入力して連結を試してみましょう
+
← 文字列を入力して連結してみましょう
// 文字列の連結（+ 演算子）
let firstName = "太郎";
let lastName  = "山田";
let fullName  = lastName + " " + firstName;
console.log(fullName); // → "山田 太郎"

// 数値を文字列に変換して連結
let age = 25;
console.log("年齢：" + age + "歳"); // → "年齢：25歳"

// テンプレートリテラル（ES6〜）バッククォートを使う
let name = "Alice";
let score = 95;
console.log(`${name}さんのスコアは${score}点です。`);
// → "Aliceさんのスコアは95点です。"

// テンプレートリテラルでは式も埋め込める
console.log(`10 + 20 = ${10 + 20}`); // → "10 + 20 = 30"

// 複数行の文字列（テンプレートリテラルなら改行もそのまま）
let message = `こんにちは、
${name}さん！
今日もよろしくお願いします。`;
console.log(message);

テンプレートリテラル vs 文字列連結

方法コード例特徴
+ 演算子 "名前：" + name + "、年齢：" + age 古い書き方。変数が多いと読みにくい
テンプレートリテラル `名前：${name}、年齢：${age}` ES6〜。読みやすく式も埋め込める。推奨
