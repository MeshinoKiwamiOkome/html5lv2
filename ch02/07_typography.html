② Web フォント

重要ポイント：@font-face でカスタムフォントを読み込めます。 Google Fonts などの CDN を使う方法と、 フォントファイルをホストする方法があります。 
/* @font-face：カスタムフォントの定義 */
@font-face {
  font-family: 'MyFont';
  src: url('fonts/myfont.woff2') format('woff2'),
       url('fonts/myfont.woff')  format('woff');
  font-weight: 400;
  font-style:  normal;
  font-display: swap; /* フォント読み込み中の表示方法 */
}

/* Google Fonts の使い方 */
/* HTML の <head> に追加 */
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="stylesheet"
      href="https://fonts.googleapis.com/css2?family=Noto+Sans+JP:wght@400;700&display=swap">

/* CSS で使用 */
body {
  font-family: 'Noto Sans JP', sans-serif;
}

/* font-display の値 */
/* auto    → ブラウザのデフォルト */
/* block   → フォント読み込みまで非表示（FOIT）*/
/* swap    → 代替フォントを表示後に切り替え（FOUT）← 推奨 */
/* fallback→ 短時間だけ非表示、その後代替フォント */
/* optional→ ネットワーク状況次第で読み込み */

/* フォントスタック（フォールバックの書き方）*/
body {
  font-family:
    'Noto Sans JP',   /* 第1候補：Web フォント */
    'Hiragino Sans',  /* 第2候補：macOS */
    'Yu Gothic',      /* 第3候補：Windows */
    sans-serif;        /* 最終フォールバック */
}

/* システムフォントスタック（Web フォント不要）*/
body {
  font-family: system-ui, -apple-system, BlinkMacSystemFont,
               'Segoe UI', Roboto, Oxygen, Ubuntu,
               Cantarell, sans-serif;
}

/* 可変フォント（Variable Fonts）*/
@font-face {
  font-family: 'Inter';
  src: url('Inter.var.woff2') format('woff2-variations');
  font-weight: 100 900; /* 範囲指定 */
}
h1 { font-variation-settings: 'wght' 650; }

③ 色の記法

重要ポイント：CSS の色指定には複数の記法があります。 hex・rgb()・hsl()・oklch() の 特徴と使い分けを理解しましょう。

🎨 カラーフォーマット コンバーター

← カラーピッカーで色を選択 
/* 色の記法一覧 */

/* 1. キーワード */
p { color: red; }
p { color: transparent; }

/* 2. HEX（16進数）*/
p { color: #553c9a; }      /* 6桁 */
p { color: #553c9aff; }    /* 8桁（アルファ付き）*/
p { color: #59a; }         /* 3桁（短縮形）*/

/* 3. RGB / RGBA */
p { color: rgb(85, 60, 154); }
p { color: rgba(85, 60, 154, 0.5); }  /* 50% 透明 */
p { color: rgb(85 60 154 / 50%); }    /* 新記法 */

/* 4. HSL / HSLA */
p { color: hsl(263, 44%, 42%); }
p { color: hsla(263, 44%, 42%, 0.8); }
/* H: 色相(0-360) S: 彩度(%) L: 明度(%) */

/* 5. oklch（モダン・知覚的に均一）*/
p { color: oklch(0.45 0.18 280); }
/* L: 明度(0-1) C: 彩度 H: 色相(0-360) */

/* 6. currentColor（現在の color 値を継承）*/
.icon { fill: currentColor; }

色の記法の使い分け

記法特徴推奨用途
#hex 短く書ける・デザインツールとの互換性高い デザインシステムの色定義
rgb() 直感的・JavaScript との連携が容易 動的な色の生成
hsl() 色相・彩度・明度で直感的に調整できる テーマカラーのバリエーション生成
oklch() 知覚的に均一・P3 色域対応 モダンなカラーシステム（CSS Level 4）

④ グラデーション

重要ポイント：CSS グラデーションは linear-gradient()・radial-gradient()・ conic-gradient() の3種類があります。 background プロパティに値として指定します。

🎨 グラデーションビルダー

135deg
0%
100%
← コントロールを操作してグラデーションを作ろう
/* linear-gradient：直線グラデーション */
.box {
  background: linear-gradient(135deg, #553c9a, #2c7a7b);
  /* 角度, 色1, 色2 */
}

/* 方向キーワード */
.box { background: linear-gradient(to right, #553c9a, #2c7a7b); }
.box { background: linear-gradient(to bottom right, red, blue); }

/* 複数色・位置指定 */
.rainbow {
  background: linear-gradient(
    to right,
    #ff0000  0%,
    #ff7700  16.6%,
    #ffff00  33.3%,
    #00ff00  50%,
    #0000ff  66.6%,
    #8b00ff  100%
  );
}

/* radial-gradient：放射グラデーション */
.box {
  background: radial-gradient(circle, #553c9a, #2c7a7b);
  /* circle / ellipse */
}

/* conic-gradient：円錐グラデーション */
.pie {
  background: conic-gradient(#553c9a 0% 30%, #2c7a7b 30% 70%, #f6ad55 70% 100%);
  border-radius: 50%; /* 円グラフ */
}

/* 複数グラデーションの重ね合わせ */
.layered {
  background:
    linear-gradient(rgba(0,0,0,0.3), rgba(0,0,0,0.3)),
    url('image.jpg') center/cover;
}

writing-mode / direction

/* writing-mode：テキストの書字方向 */
.horizontal { writing-mode: horizontal-tb; } /* 横書き（デフォルト）*/
.vertical-rl{ writing-mode: vertical-rl; }   /* 縦書き（右から左）*/
.vertical-lr{ writing-mode: vertical-lr; }   /* 縦書き（左から右）*/

/* direction：テキストの流れる方向 */
.rtl { direction: rtl; } /* 右から左（アラビア語など）*/
.ltr { direction: ltr; } /* 左から右（デフォルト）*/

/* text-orientation：縦書き時の文字の向き */
.mixed  { text-orientation: mixed; }   /* 欧文は横向き */
.upright{ text-orientation: upright; } /* 欧文も縦向き */

📌 まとめ

カテゴリプロパティ重要ポイント
フォント font-family フォールバックを必ず指定・system-ui で OS フォント
font-size rem 推奨・16px = 1rem（デフォルト）
font-weight 100〜900・bold = 700・normal = 400
line-height 単位なし推奨（1.5〜1.8 が読みやすい）
@font-face font-display: swap でパフォーマンス向上
色 #hex 6桁・8桁（アルファ付き）・3桁（短縮形）
rgb() / rgba() 0〜255 の3値・アルファは 0〜1
hsl() 色相(0-360)・彩度(%)・明度(%)・テーマ生成に便利
oklch() 知覚的均一・モダンなカラーシステム
グラデーション linear-gradient() 角度または方向キーワード・複数色指定可
radial-gradient() circle / ellipse・中心から放射状
conic-gradient() 円錐状・円グラフに活用
