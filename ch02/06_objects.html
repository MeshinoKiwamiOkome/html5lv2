🧩 オブジェクトの基本

// オブジェクトリテラル
const user = {
  name: 'Alice',
  age:  30,
  greet() {                       // メソッド短縮記法（ES6）
    return `Hello, ${this.name}`;
  }
};

// プロパティアクセス
console.log(user.name);          // "Alice"（ドット記法）
console.log(user['age']);        // 30（ブラケット記法）

// プロパティの追加・削除
user.email = 'alice@example.com'; // 追加
delete user.age;                    // 削除

// スプレッド構文・分割代入
const copy = { ...user, role: 'admin' }; // シャローコピー＋追加
const { name, email = 'none' } = user;    // 分割代入（デフォルト値付き）

// 計算プロパティ名（ES6）
const key = 'score';
const obj = { [key]: 100 };       // { score: 100 }

🛠 Object 静的メソッド

メソッド説明戻り値
Object.keys(obj)列挙可能な自身のキー一覧文字列の配列
Object.values(obj)列挙可能な自身の値一覧値の配列
Object.entries(obj)列挙可能な自身の [key, value] ペア一覧[key,val]の配列
Object.assign(target, ...src)プロパティをコピー（シャロー）target オブジェクト
Object.freeze(obj)オブジェクトを凍結（変更不可）凍結されたオブジェクト
Object.create(proto)指定プロトタイプでオブジェクト生成新しいオブジェクト
Object.defineProperty(obj, key, desc)プロパティ記述子を定義obj
Object.fromEntries(iter)[key,val] の配列からオブジェクト生成新しいオブジェクト
Object.hasOwn(obj, key)自身のプロパティか確認（ES2022）boolean
 
const obj = { a: 1, b: 2, c: 3 };

console.log(Object.keys(obj));    // ["a", "b", "c"]
console.log(Object.values(obj));  // [1, 2, 3]
console.log(Object.entries(obj)); // [["a",1],["b",2],["c",3]]

// Object.assign：シャローコピー
const merged = Object.assign({}, obj, { d: 4 });
console.log(merged); // { a:1, b:2, c:3, d:4 }

// Object.freeze：変更不可にする
const frozen = Object.freeze({ x: 1 });
frozen.x = 99; // 無視される（strictモードではTypeError）
console.log(frozen.x); // 1（変更されない）

// entries → fromEntries で変換
const doubled = Object.fromEntries(
  Object.entries(obj).map(([k, v]) => [k, v * 2])
);
console.log(doubled); // { a:2, b:4, c:6 }

🔗 prototype とプロトタイプチェーン

JavaScriptのすべてのオブジェクトは [[Prototype]]（内部スロット）を持ち、 プロパティを参照するとき自身に見つからなければプロトタイプチェーンを辿って探します。 

// コンストラクタ関数とprototype
function Animal(name) {
  this.name = name;
}
Animal.prototype.speak = function() {
  return `${this.name} speaks`;
};

const dog = new Animal('Dog');
console.log(dog.speak());         // "Dog speaks"
console.log(dog.hasOwnProperty('name'));  // true（自身のプロパティ）
console.log(dog.hasOwnProperty('speak')); // false（prototypeのプロパティ）

// プロトタイプチェーンの確認
console.log(Object.getPrototypeOf(dog) === Animal.prototype); // true
console.log(dog instanceof Animal);  // true
console.log(dog instanceof Object);  // true（全てObjectを継承）

🏛 class 構文

ES6 で導入された class 構文は、プロトタイプベースの継承を分かりやすく書くための糖衣構文です。

class Animal {
  // プライベートフィールド（ES2022）
  #sound = '...';

  constructor(name, sound) {
    this.name  = name;
    this.#sound = sound;
  }

  // インスタンスメソッド
  speak() {
    return `${this.name}: ${this.#sound}`;
  }

  // getter / setter
  get info() {
    return `[${this.name}]`;
  }
  set nickname(val) {
    this.name = val;
  }

  // static メソッド（インスタンスではなくクラスから呼ぶ）
  static create(name, sound) {
    return new Animal(name, sound);
  }
}

// 継承
class Dog extends Animal {
  constructor(name) {
    super(name, 'Woof'); // 親のconstructorを呼ぶ（必須）
    this.tricks = [];
  }

  learn(trick) {
    this.tricks.push(trick);
  }

  // メソッドのオーバーライド
  speak() {
    return super.speak() + ' 🐕'; // 親メソッドを呼ぶ
  }
}

const dog = new Dog('Rex');
console.log(dog.speak());          // "Rex: Woof 🐕"
console.log(dog.info);              // "[Rex]"（getter）
console.log(dog instanceof Dog);    // true
console.log(dog instanceof Animal); // true（継承関係）
console.log(Animal.create('Cat', 'Meow').speak()); // "Cat: Meow"
構文説明
constructor()インスタンス生成時に呼ばれる初期化メソッド
extendsクラスの継承
super()親クラスのコンストラクタを呼ぶ（extends 時は必須）
super.method()親クラスのメソッドを呼ぶ
staticクラスメソッド・プロパティ（インスタンスからは呼べない）
get / setアクセサプロパティ（getter / setter）
#fieldプライベートフィールド（クラス外からアクセス不可）
⚠️ 試験ポイント：extends 使用時の super()
extends を使ったサブクラスの constructor 内では、 this を使う前に必ず super() を呼ぶ必要があります。 呼ばないと ReferenceError になります。
