① セレクタの種類

重要ポイント：CSS セレクタはスタイルを適用する要素を指定する仕組みです。 セレクタの種類と詳細度（Specificity）を正確に理解することが試験の核心です。

セレクタ一覧

種類例説明詳細度
全称セレクタ * すべての要素 0,0,0
タイプセレクタ p, div タグ名で指定 0,0,1
クラスセレクタ .btn class 属性で指定 0,1,0
ID セレクタ #header id 属性で指定 1,0,0
属性セレクタ [type="text"] 属性値で指定 0,1,0
疑似クラス :hover, :nth-child() 状態・位置で指定 0,1,0
疑似要素 ::before, ::after 仮想要素を生成 0,0,1
インラインスタイル style="..." HTML 要素に直接記述 1,0,0,0
!important color: red !important 詳細度を超えて優先 最優先

結合子（Combinator）

結合子例説明
子孫結合子（スペース） div p div の子孫すべての p
子結合子（>） div > p div の直接の子の p のみ
隣接兄弟結合子（+） h1 + p h1 の直後の兄弟 p
一般兄弟結合子（~） h1 ~ p h1 以降のすべての兄弟 p

② 詳細度（Specificity）

重要ポイント：詳細度は (ID, クラス/属性/疑似クラス, タイプ/疑似要素) の3つの数値で表されます。数値が大きいほど優先されます。 同じ詳細度の場合は後から書いたスタイルが優先されます（カスケード）。

🧮 詳細度カリキュレーター

詳細度の計算ルール

/* 詳細度の計算例 */
/* 形式：(ID, クラス/属性/疑似クラス, タイプ/疑似要素) */

*              { } /* (0, 0, 0) */
p              { } /* (0, 0, 1) */
.class         { } /* (0, 1, 0) */
#id            { } /* (1, 0, 0) */
p.class        { } /* (0, 1, 1) */
#id .class p   { } /* (1, 1, 1) */
div p:hover    { } /* (0, 1, 2) */
ul li::before  { } /* (0, 0, 3) */

/* 詳細度の比較 */
#id { color: red; }    /* (1,0,0) ← 勝ち */
.a .b .c .d .e .f .g .h .i .j .k { color: blue; }
/* (0,11,0) ← 負け（クラスがいくら多くても ID に勝てない）*/

③ 疑似クラスと疑似要素

重要ポイント：疑似クラス（:）は要素の状態や位置を指定し、 疑似要素（::）は仮想的な要素を生成します。 ::before と ::after は content プロパティが必須です。

主要な疑似クラス

  • 最初の項目（:first-child）
  • 2番目の項目
  • 3番目の項目（:nth-child(3)）
  • 4番目の項目（:nth-child(even)）
  • 最後の項目（:last-child）
/* 状態系 */
a:hover    { color: blue; }    /* マウスオーバー */
a:visited  { color: purple; } /* 訪問済み */
a:active   { color: red; }     /* クリック中 */
input:focus{ outline: 2px solid blue; } /* フォーカス */
input:disabled{ opacity: 0.5; }  /* 無効 */
input:checked{ accent-color: blue; } /* チェック済み */

/* 構造系 */
li:first-child  { font-weight: bold; }
li:last-child   { font-style: italic; }
li:nth-child(2) { color: blue; }    /* 2番目 */
li:nth-child(odd)  { background: #f0f0f0; } /* 奇数 */
li:nth-child(even) { background: #fff; }    /* 偶数 */
li:nth-child(3n+1) { color: red; }     /* 1,4,7,10... */

/* 否定 */
p:not(.special){ color: gray; }  /* .special 以外の p */

疑似要素（::）

/* ::before / ::after：仮想要素を生成 */
/* content プロパティが必須！ */
.badge::before {
  content: "NEW ";  /* テキストを前に挿入 */
  color: red;
  font-weight: bold;
}

.price::after {
  content: " 円";  /* テキストを後に挿入 */
}

/* content に attr() で属性値を使う */
a::after {
  content: " (" attr(href) ")";
}

/* ::first-line / ::first-letter */
p::first-line   { font-weight: bold; }
p::first-letter { font-size: 2em; float: left; }

/* ::selection：テキスト選択時 */
::selection {
  background: #553c9a;
  color: #fff;
}

/* ::placeholder：input のプレースホルダー */
input::placeholder { color: #a0aec0; }
::before / ::after の動作を確認しよう

:nth-child() の式

← 式を入力して適用してみましょう
/* :nth-child() の式パターン */
:nth-child(1)    /* 1番目のみ */
:nth-child(odd)  /* 奇数（1,3,5,...）= :nth-child(2n+1) */
:nth-child(even) /* 偶数（2,4,6,...）= :nth-child(2n) */
:nth-child(3n)   /* 3の倍数（3,6,9,...）*/
:nth-child(3n+1) /* 1,4,7,10,...（3n+1）*/
:nth-child(-n+3) /* 最初の3つ（1,2,3）*/

/* :nth-of-type()：同じタグ型の中での順番 */
p:nth-of-type(2) /* 2番目の p 要素 */

/* :nth-child vs :nth-of-type の違い */
/* HTML: <div><h2>...</h2><p>1</p><p>2</p></div> */
p:nth-child(2)      /* → p の中で2番目の子 = "1" */
p:nth-of-type(2)    /* → p 要素の中で2番目 = "2" */

属性セレクタ

/* 属性セレクタのパターン */
[href]           /* href 属性を持つ要素 */
[type="text"]    /* type が "text" に完全一致 */
[class~="btn"]   /* class に "btn" を含む（スペース区切り）*/
[lang|="en"]     /* lang が "en" または "en-" で始まる */
[href^="https"]  /* href が "https" で始まる */
[href$=".pdf"]   /* href が ".pdf" で終わる */
[href*="google"] /* href に "google" を含む */

/* 実用例 */
input[type="submit"] { background: #553c9a; }
a[href^="https"]::after {
  content: " 🔒"; /* HTTPS リンクにアイコン */
}
a[href$=".pdf"]::after {
  content: " 📄"; /* PDF リンクにアイコン */
}

④ カスケードと継承

重要ポイント：同じ要素に複数のスタイルが適用される場合、 ① !important → ② インラインスタイル → ③ 詳細度 → ④ 記述順序 の優先順位で決まります（カスケード）。 
/* カスケードの優先順位（高い順）*/
/* 1. !important（最優先）*/
p { color: red !important; }   /* 最優先 */

/* 2. インラインスタイル */
/* <p style="color: blue"> ← 詳細度 (1,0,0,0) */

/* 3. 詳細度が高いセレクタ */
#id .class p { color: green; }  /* (1,1,1) */
.class p     { color: yellow; } /* (0,1,1) */

/* 4. 同じ詳細度なら後から書いたほうが優先 */
p { color: orange; }  /* ← こちらが適用される */
p { color: purple; }  /* ← 後から書いたので優先 */

/* 継承されるプロパティ（一部）*/
/* color, font-*, line-height, text-align など */
body { font-family: sans-serif; } /* 子要素に継承される */

/* 継承されないプロパティ */
/* margin, padding, border, background など */

/* inherit / initial / unset */
p { color: inherit; }  /* 親から継承を強制 */
p { color: initial; } /* ブラウザのデフォルト値に戻す */
p { color: unset; }   /* 継承可能なら inherit、そうでなければ initial */
