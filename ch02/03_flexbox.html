① 主軸（Main Axis）と交差軸（Cross Axis）

重要ポイント：Flexbox の核心は「主軸」と「交差軸」の理解です。 flex-direction で主軸の方向を決め、 justify-content は主軸方向、 align-items は交差軸方向の配置を制御します。

row → 縦方向
column → 横方向

② コンテナプロパティ

重要ポイント：Flexbox のプロパティは「コンテナ（親要素）に設定するもの」と 「アイテム（子要素）に設定するもの」に分かれます。 まずコンテナプロパティを完全に理解しましょう。

コンテナプロパティ一覧

プロパティ値説明
flex-direction row / column / row-reverse / column-reverse 主軸の方向
justify-content flex-start / center / space-between / space-around / space-evenly 主軸方向の配置
align-items stretch / flex-start / center / flex-end / baseline 交差軸方向の配置（1行）
flex-wrap nowrap / wrap / wrap-reverse 折り返し
align-content flex-start / center / space-between など 複数行の交差軸配置
gap 8px / 8px 16px アイテム間の隙間
flex-flow row wrap flex-direction + flex-wrap のショートハンド

③ アイテムプロパティ

重要ポイント：flex-grow・flex-shrink・flex-basis は アイテムの伸縮を制御します。 flex: 1 は flex-grow: 1; flex-shrink: 1; flex-basis: 0% の ショートハンドです。

/* flex-grow：余白をどれだけ伸びて埋めるか（デフォルト: 0）*/
.item-a { flex-grow: 1; } /* 余白を1の比率で取得 */
.item-b { flex-grow: 2; } /* item-a の2倍の余白を取得 */

/* flex-shrink：スペース不足時にどれだけ縮むか（デフォルト: 1）*/
.item    { flex-shrink: 1; } /* 縮む（デフォルト）*/
.no-shrink{ flex-shrink: 0; } /* 縮まない */

/* flex-basis：アイテムの基本サイズ（デフォルト: auto）*/
.item { flex-basis: 200px; } /* 基本幅 200px */
.item { flex-basis: 0; }     /* コンテンツ幅を無視 */

/* flex ショートハンド */
.item { flex: 1; }
/* = flex-grow:1; flex-shrink:1; flex-basis:0% */

.item { flex: 0 0 200px; }
/* = flex-grow:0; flex-shrink:0; flex-basis:200px */
/* → 固定幅 200px（伸縮しない）*/

/* align-self：個別の交差軸配置 */
.special { align-self: flex-end; }

/* order：表示順序（デフォルト: 0）*/
.first { order: -1; } /* 先頭に移動 */
.last  { order: 1; }  /* 末尾に移動 */

④ よくある Flexbox レイアウトパターン

完全中央揃え

完全中央揃え 
/* 完全中央揃え（最もシンプルな方法）*/
.container {
  display:         flex;
  justify-content: center;  /* 主軸（横）中央 */
  align-items:     center;  /* 交差軸（縦）中央 */
  height:          100vh;   /* 画面全体の高さ */
}

/* または margin: auto を使う方法 */
.container { display: flex; }
.child     { margin: auto; }

📌 まとめ

対象プロパティ主な値説明
コンテナ display flex / inline-flex Flexbox を有効化
flex-direction row / column 主軸の方向
justify-content center / space-between 主軸方向の配置
align-items center / stretch 交差軸方向の配置
flex-wrap wrap / nowrap 折り返し
gap 8px / 8px 16px アイテム間の隙間
アイテム flex-grow 0 / 1 / 2 余白を埋める比率
flex-shrink 0 / 1 縮む比率（0で縮まない）
flex-basis auto / 200px 基本サイズ
align-self center / flex-end 個別の交差軸配置
order -1 / 0 / 1 表示順序
