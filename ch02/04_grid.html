① Grid の基本とグリッド線

重要ポイント：CSS Grid は行（row）と列（column）の2次元レイアウトシステムです。 グリッド線は1から始まる番号で管理され、 grid-column・grid-row でアイテムの配置範囲を指定します。

グリッド線の番号体系

/* Grid の基本 */
.container {
  display:                  grid;
  grid-template-columns:    200px 1fr 1fr;  /* 3列 */
  grid-template-rows:       100px auto 60px; /* 3行 */
  gap:                      16px;
}

/* グリッド線番号でアイテムを配置 */
.item {
  grid-column: 1 / 3;  /* 列線1〜3（2列分）*/
  grid-row:    1 / 2;  /* 行線1〜2（1行分）*/
}

/* span を使った指定 */
.item {
  grid-column: 1 / span 2; /* 1列目から2列分 */
  grid-row:    span 2;     /* 現在位置から2行分 */
}

/* 負の番号（末尾から）*/
.full-width {
  grid-column: 1 / -1; /* 1列目から最後の列まで */
}

fr 単位と repeat()

/* fr（fraction）：残りスペースの比率 */
.container {
  grid-template-columns: 1fr 2fr 1fr;
  /* 全体を4等分し、左1/4・中2/4・右1/4 */
}

/* repeat()：繰り返し */
.container {
  grid-template-columns: repeat(3, 1fr);
  /* = 1fr 1fr 1fr（3等分）*/
}

/* 固定幅 + 可変幅の組み合わせ */
.layout {
  grid-template-columns: 200px 1fr;
  /* サイドバー固定 + メインコンテンツ可変 */
}

/* minmax()：最小・最大サイズを指定 */
.container {
  grid-template-columns: repeat(3, minmax(200px, 1fr));
  /* 最小200px、最大均等分割 */
}

/* auto-fill：コンテナ幅に収まるだけ列を作る */
.auto-fill {
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
}

/* auto-fit：空のトラックを折り畳む */
.auto-fit {
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
}

③ grid-area と名前付きエリア

重要ポイント：grid-template-areas で名前付きエリアを定義し、 grid-area でアイテムを配置できます。 レイアウトの意図が視覚的に分かりやすくなります。

名前付きエリアのビジュアル

header
sidebar
main
footer
/* 名前付きエリアの定義 */
.container {
  display: grid;
  grid-template-areas:
    "header  header  header"
    "sidebar main    main"
    "sidebar main    main"
    "footer  footer  footer";
  grid-template-columns: 160px 1fr 1fr;
  grid-template-rows:    50px 1fr 1fr 50px;
  gap: 8px;
}

/* 各アイテムにエリア名を割り当て */
.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }
.footer  { grid-area: footer; }

/* "." で空白セルを表現 */
.container {
  grid-template-areas:
    "header header ."
    "main   main   sidebar";
}

span によるセル結合ビジュアル

/* grid-column / grid-row でアイテムを配置 */

/* 方法1：開始線 / 終了線 */
.item {
  grid-column: 1 / 3;  /* 列線1から列線3（2列分）*/
  grid-row:    1 / 3;  /* 行線1から行線3（2行分）*/
}

/* 方法2：span キーワード */
.item {
  grid-column: 2 / span 3; /* 2列目から3列分 */
  grid-row:    span 2;     /* 現在位置から2行分 */
}

/* 方法3：grid-area ショートハンド */
/* grid-area: row-start / col-start / row-end / col-end */
.item {
  grid-area: 1 / 1 / 3 / 3;
  /* 行1〜3、列1〜3（2×2のブロック）*/
}

/* 負の番号で末尾から指定 */
.full-row {
  grid-column: 1 / -1; /* 全列にまたがる */
}

⑤ Flexbox vs Grid の使い分け

観点FlexboxCSS Grid
次元 1次元（行 または 列） 2次元（行 と 列）
得意なこと コンポーネント内の配置・ナビゲーション ページ全体のレイアウト・複雑な配置
アイテムのサイズ コンテンツに応じて柔軟に変化 グリッドトラックに基づく厳密な配置
折り返し flex-wrap で折り返し可能 auto-fill/auto-fit で自動配置
典型的な用途 ボタングループ・カードの中身・ナビ ページレイアウト・ダッシュボード・ギャラリー
💡 実践的な使い分け：
・Flexbox：「これらの要素を横に並べたい」→ 1次元の配置
・Grid：「このページを3列レイアウトにしたい」→ 2次元の配置
・両方を組み合わせることも多い（Grid でページ構造、Flexbox でコンポーネント内部）

📌 まとめ

対象プロパティ説明
コンテナ display: grid Grid を有効化

grid-template-rows 行の定義

gap セル間の隙間
justify-items / align-items セル内のアイテム配置
アイテム grid-column 列方向の配置（開始線/終了線 or span）
grid-row 行方向の配置
grid-area 名前付きエリアへの配置 or 4値ショートハンド
justify-self / align-self 個別のセル内配置
便利な値 fr 残りスペースの比率単位

span N N個のトラックにまたがる
