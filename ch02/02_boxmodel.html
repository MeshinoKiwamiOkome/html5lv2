① ボックスモデルの構造

重要ポイント：すべての HTML 要素は content・padding・border・margin の 4層で構成される「ボックス」として扱われます。 box-sizing の値によって width/height の計算方法が変わります。

ボックスモデルの4層構造

margin（外側の余白）
border（枠線）
padding（内側の余白）
content（内容）
width × height

/* ボックスモデルの基本 */
.box {
  width:   200px;   /* コンテンツ幅 */
  height:  100px;   /* コンテンツ高さ */
  padding: 20px;    /* 内側の余白（上下左右） */
  border:  2px solid #333; /* 枠線 */
  margin:  16px;    /* 外側の余白 */
}

/* content-box（デフォルト）*/
/* 実際の幅 = width + padding×2 + border×2 */
/* 200 + 20×2 + 2×2 = 244px */

/* border-box（推奨）*/
/* 実際の幅 = width（padding と border を含む）*/
/* 200px のまま */
.box { box-sizing: border-box; }

/* 全要素に border-box を適用（モダンなリセット）*/
*, *::before, *::after {
  box-sizing: border-box;
}

② margin と padding

重要ポイント：margin は要素の外側の余白、padding は内側の余白です。 値の指定方法（1〜4個）と margin の相殺（マージンの折り畳み）を理解しましょう。

値の個数による指定方法

/* 値1個：上下左右すべて同じ */
p { margin: 16px; }
/* → 上:16px 右:16px 下:16px 左:16px */

/* 値2個：上下 / 左右 */
p { margin: 8px 16px; }
/* → 上下:8px 左右:16px */

/* 値3個：上 / 左右 / 下 */
p { margin: 8px 16px 24px; }
/* → 上:8px 左右:16px 下:24px */

/* 値4個：上 / 右 / 下 / 左（時計回り）*/
p { margin: 8px 16px 24px 32px; }
/* → 上:8px 右:16px 下:24px 左:32px */

/* 個別指定 */
p {
  margin-top:    8px;
  margin-right:  16px;
  margin-bottom: 24px;
  margin-left:   32px;
}

/* 論理プロパティ（CSS Logical Properties）*/
p {
  margin-block:  8px;  /* 上下（書字方向に応じて）*/
  margin-inline: 16px; /* 左右 */
}

margin の相殺（マージンの折り畳み）

margin の相殺とは：隣接するブロック要素の上下 margin は 大きいほうの値に統合されます（加算されない）。 左右の margin は相殺されません。
要素A（margin-bottom: 30px）
要素B（margin-top: 20px）
↑ 実際の間隔は 30px + 20px = 50px ではなく、max(30px, 20px) = 30px

/* margin の相殺が起きる条件 */
/* 1. 隣接する兄弟要素の上下 margin */
.box-a { margin-bottom: 30px; }
.box-b { margin-top:    20px; }
/* 間隔 = max(30, 20) = 30px（相殺！）*/

/* 相殺を防ぐ方法 */
.parent {
  overflow: hidden;  /* BFC を作成 */
  /* または */
  padding-top: 1px;  /* padding を追加 */
  /* または */
  border-top: 1px solid transparent;
}

/* Flexbox / Grid コンテナ内では相殺が起きない */
.flex-container { display: flex; flex-direction: column; }

margin: auto による中央揃え

/* margin: auto で水平中央揃え */
.centered {
  width:  300px;   /* 幅が必要！ */
  margin: 0 auto; /* 上下:0 左右:auto */
}

/* auto の動作 */
/* margin-left: auto → 左に最大の余白 → 右寄せ */
.right-align { margin-left: auto; }

/* Flexbox での margin: auto */
.flex-container { display: flex; }
.push-right     { margin-left: auto; } /* 右端に押し出す */

/* ⚠️ 垂直方向の auto は通常効かない */
/* Flexbox / Grid では垂直 auto も有効 */
.flex-child { margin: auto; } /* 上下左右中央 */

③ display プロパティ

重要ポイント：display は要素の表示形式を決定します。 block・inline・inline-block の違いと、 none・flex・grid の使い分けが重要です。

display 値の比較

値幅・高さ改行説明
block ✅ 指定可 前後に改行 div, p, h1〜h6 のデフォルト
inline ❌ 指定不可 改行なし span, a, strong のデフォルト
inline-block ✅ 指定可 改行なし inline + block の特性を持つ
none - - 要素を非表示（スペースも消える）
flex ✅ 指定可 前後に改行 Flexbox コンテナ
grid ✅ 指定可 前後に改行 Grid コンテナ
contents - - 要素自体のボックスを消す（子要素は残る）

④ border プロパティ

/* border のショートハンド */
.box { border: 2px solid #333; }
/* 太さ スタイル 色 の順 */

/* border-style の種類 */
.solid  { border: 2px solid #333; }   /* 実線 */
.dashed { border: 2px dashed #333; }  /* 破線 */
.dotted { border: 2px dotted #333; }  /* 点線 */
.double { border: 4px double #333; }  /* 二重線 */
.none   { border: none; }             /* なし */

/* border-radius：角丸 */
.rounded { border-radius: 8px; }
.circle  { border-radius: 50%; }  /* 正方形に適用で円 */
.pill    { border-radius: 9999px; } /* カプセル型 */

/* 個別指定 */
.box {
  border-top:    2px solid red;
  border-right:  2px solid blue;
  border-bottom: 2px solid green;
  border-left:   2px solid orange;
}

/* outline：border に似るがボックスモデルに影響しない */
button:focus { outline: 2px solid #553c9a; }

📌 まとめ

プロパティ説明重要ポイント
width / height コンテンツ領域のサイズ box-sizing によって実際のサイズが変わる
padding 内側の余白 背景色が適用される・4方向個別指定可
border 枠線 太さ・スタイル・色の3値
margin 外側の余白 隣接する上下 margin は相殺される
box-sizing: content-box デフォルト 実幅 = width + padding×2 + border×2
box-sizing: border-box 推奨 実幅 = width（padding・border を含む）
display: block ブロック要素 幅・高さ指定可・前後改行
display: inline インライン要素 幅・高さ指定不可・改行なし
display: inline-block インラインブロック 幅・高さ指定可・改行なし
display: none 非表示 スペースも消える（visibility: hidden と違う）
