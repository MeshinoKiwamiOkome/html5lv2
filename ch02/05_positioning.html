① position の5つの値

重要ポイント：position プロパティは要素の配置方法を決定します。 top・right・bottom・left で 位置を指定しますが、static ではこれらは無効です。

position の値と特徴

値 基準点 通常フロー top/left 等 主な用途
static - ✅ 維持 ❌ 無効 デフォルト（通常の配置）
relative 元の位置 ✅ 維持（スペース残る） ✅ 有効 微調整・absolute の基準点
absolute 最近傍の positioned 祖先 ❌ 外れる ✅ 有効 ツールチップ・バッジ・ドロップダウン
fixed ビューポート ❌ 外れる ✅ 有効 固定ヘッダー・FABボタン・モーダル
sticky スクロール親要素 ✅ 維持（スクロール後固定） ✅ 有効（閾値） スティッキーヘッダー・サイドバー

② 各 position の詳細

position: relative

relative の特徴：元の位置を基準に移動します。 元のスペースは残ったままです。 また、absolute の子要素の基準点になります。
通常要素 A
relative
top:20px left:10px
通常要素 C
↑ relative 要素は元のスペースを保持したまま移動（C は元の位置に配置） 
/* relative：元の位置から相対的に移動 */
.box {
  position: relative;
  top:  20px;  /* 元の位置から下に20px */
  left: 10px;  /* 元の位置から右に10px */
}
/* 元のスペースは残る → 他の要素に影響しない */

/* 主な用途：absolute の基準点を作る */
.parent {
  position: relative; /* ← これが基準点 */
}
.child {
  position: absolute;
  top: 0; right: 0; /* 親の右上に配置 */
}

position: absolute

absolute の特徴：通常フローから外れ、 最も近い position: static 以外の祖先要素を基準に配置されます。 基準となる祖先がない場合は <html> が基準になります。
親要素（position: relative）
通常要素（フロー内）
absolute
top:8px right:8px
次の通常要素（absolute の分スペースなし） 
/* absolute：最近傍の positioned 祖先を基準に配置 */
.parent {
  position: relative; /* 基準点を作る（重要！）*/
}
.badge {
  position: absolute;
  top:   -8px;
  right: -8px;
  /* 親の右上角に配置 */
}

/* 上下左右中央揃え（古典的な方法）*/
.centered {
  position:  absolute;
  top:        50%;
  left:       50%;
  transform: translate(-50%, -50%);
}

/* 全面に広げる */
.overlay {
  position: absolute;
  inset: 0; /* top:0 right:0 bottom:0 left:0 */
}

position: fixed

fixed の特徴：ビューポート（画面）を基準に配置されます。 スクロールしても位置が変わりません。 通常フローから外れるため、他の要素に影響しません。 
/* fixed：ビューポートを基準に固定 */
.fixed-header {
  position: fixed;
  top:   0;
  left:  0;
  right: 0;
  z-index: 100;
  /* スクロールしても常に画面上部に固定 */
}

.fab-button {
  position: fixed;
  bottom: 24px;
  right:  24px;
  /* 画面右下に固定されるボタン */
}

/* fixed 要素の下にコンテンツが隠れないよう padding を追加 */
body {
  padding-top: 60px; /* ヘッダーの高さ分 */
}
⚠️ fixed の注意点： transform・filter・perspective が設定された 祖先要素がある場合、ビューポートではなくその要素が基準になります。

position: sticky

sticky の特徴：通常は relative と同様に動作し、 スクロールで指定した閾値（top など）に達すると fixed のように固定されます。 親要素の範囲内でのみ固定されます。

.sticky-header {

③ z-index と重なり順序

重要ポイント：z-index は position: static 以外の要素に 有効です。値が大きいほど前面に表示されます。 z-index は同じスタッキングコンテキスト内でのみ比較されます。

🎮 z-index インタラクティブデモ

z-index: 1
Box B
z-index: 2
Box C
z-index: 3
ボックスをクリックして z-index を変更しよう
← ボックスをクリックするか、値を入力して適用してみましょう
/* z-index：重なり順序（大きいほど前面）*/
.modal-overlay { z-index: 100; }
.modal         { z-index: 101; }
.tooltip       { z-index: 200; }

/* z-index は position: static 以外で有効 */
.no-effect {
  position: static;
  z-index:  999; /* ← 効かない！ */
}

/* スタッキングコンテキスト */
/* 以下のプロパティが新しいスタッキングコンテキストを作る */
/* position + z-index（auto 以外）*/
/* opacity < 1 */
/* transform, filter, perspective */
/* isolation: isolate */

/* z-index の管理（CSS 変数で一元管理）*/
:root {
  --z-dropdown: 100;
  --z-modal:    200;
  --z-tooltip:  300;
}

④ float と clear

重要ポイント：float は要素を左右に浮かせてテキストを回り込ませます。 現代では Flexbox・Grid が推奨されますが、 試験では clear と clearfix の知識が問われます。 
/* float：要素を左右に浮かせる */
.image {
  float: left;   /* 左に浮かせる */
  margin: 0 16px 16px 0;
}

/* clear：float の回り込みを解除 */
.footer {
  clear: both; /* 左右両方の float を解除 */
}

/* clearfix：親要素の高さ崩れを防ぐ */
.clearfix::after {
  content:  "";
  display:  block;
  clear:    both;
}

/* 現代的な代替手段 */
.container {
  display: flow-root; /* BFC を作成して高さ崩れを防ぐ */
}

⑤ inset と便利なショートハンド

/* inset：top/right/bottom/left のショートハンド */
.overlay {
  position: absolute;
  inset: 0;
  /* = top:0; right:0; bottom:0; left:0 */
}

.box {
  inset: 10px 20px;
  /* = top:10px; right:20px; bottom:10px; left:20px */
}

/* 中央揃えのモダンな書き方 */
.modal {
  position: fixed;
  inset:    0;             /* 全面 */
  display:  flex;
  align-items:     center;
  justify-content: center;
}

/* positioned 要素の全面オーバーレイパターン */
.card         { position: relative; }
.card-overlay {
  position: absolute;
  inset:    0;
  background: rgba(0,0,0,0.5);
  opacity:    0;
  transition: opacity 0.3s;
}
.card:hover .card-overlay { opacity: 1; }

📌 まとめ

プロパティ/値説明重要ポイント
position: static デフォルト top/left 等は無効
position: relative 元の位置から相対移動 スペースは残る・absolute の基準点になる
position: absolute 最近傍 positioned 祖先を基準 フローから外れる・親に relative が必要
position: fixed ビューポートを基準 スクロールしても固定・フローから外れる
position: sticky スクロールで固定 top 等の閾値が必要・親要素の範囲内のみ
z-index 重なり順序 static 以外で有効・スタッキングコンテキストに注意
float 左右に浮かせる テキスト回り込み・clearfix で高さ崩れ防止
clear float の解除 both で左右両方解除
inset top/right/bottom/left の一括指定 inset: 0 で全面に広げる
