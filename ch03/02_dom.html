① DOM とは何か

DOMのツリー構造

DOM（Document Object Model）は、HTMLドキュメントをツリー状のオブジェクト構造として表現したものです。 ブラウザはHTMLを解析してDOMツリーを構築し、JavaScriptはこのDOMを通じてページの内容・構造・スタイルを動的に変更できます。

DOMツリーの各要素はノード（Node）と呼ばれます。 主なノードの種類は「要素ノード（Element）」「テキストノード（Text）」「属性ノード（Attr）」「コメントノード（Comment）」です。

ノードの種類説明例
Element ノードHTML要素そのもの<div> <p> <ul>
Text ノード要素内のテキスト"Hello World"
Attr ノード要素の属性id="main" class="btn"
Comment ノードHTMLコメント<!-- コメント -->
Document ノードドキュメント全体のルートdocument
📌 試験頻出：document オブジェクトの位置づけ document はDOMツリーのルートノードであり、window.document としてアクセスできます。 document.documentElement は <html> 要素、 document.body は <body> 要素を指します。

② 要素の取得

取得メソッドの種類と特徴

メソッド返り値動的/静的特徴
getElementById(id)Element / null—最速。IDで1要素取得
getElementsByClassName(cls)HTMLCollection動的クラス名で複数取得
getElementsByTagName(tag)HTMLCollection動的タグ名で複数取得
querySelector(sel)Element / null—CSSセレクタで最初の1つ
querySelectorAll(sel)NodeList静的CSSセレクタで全て取得
 
// ID で取得（最も高速）
const el = document.getElementById('myId');

// CSSセレクタで最初の1要素
const btn = document.querySelector('.btn-primary');
const h1  = document.querySelector('h1');
const inp = document.querySelector('input[type="text"]');

// CSSセレクタで全要素（NodeList）
const items = document.querySelectorAll('.list-item');
items.forEach(item => console.log(item.textContent));

// クラス名で取得（HTMLCollection：動的）
const cards = document.getElementsByClassName('card');

// 子孫要素に絞って検索（要素.querySelector も使える）
const nav  = document.getElementById('nav');
const link = nav.querySelector('a.active'); // nav 内の .active なリンク
⚠️ NodeList と HTMLCollection の違い querySelectorAll が返す NodeList は静的（取得後にDOMが変わっても反映されない）。 getElementsByClassName などが返す HTMLCollection は動的（DOM変更がリアルタイムに反映される）。 詳しくは 3-4 セレクタAPI で解説します。

③ DOM の操作

テキスト・スタイル・クラスの変更

const el = document.querySelector('#target');

// テキスト変更
el.textContent = '新しいテキスト';  // 安全（HTMLエスケープ）
el.innerHTML   = '<strong>太字</strong>'; // HTML解釈（XSS注意！）

// スタイル変更（インラインスタイル）
el.style.color           = 'red';
el.style.backgroundColor = '#f0f4f8';  // キャメルケース
el.style.fontSize        = '18px';

// classList API（推奨）
el.classList.add('active');         // クラス追加
el.classList.remove('hidden');      // クラス削除
el.classList.toggle('open');        // あれば削除・なければ追加
el.classList.contains('active');   // 存在確認 → true/false
el.classList.replace('old', 'new'); // クラス置換

// 属性操作
el.setAttribute('data-id', '42');   // 属性を設定
el.getAttribute('data-id');         // → '42'
el.removeAttribute('disabled');     // 属性を削除
el.hasAttribute('hidden');          // 存在確認 → true/false

要素の生成・追加・削除

// 要素の生成
const li = document.createElement('li');
li.textContent = '新しいアイテム';
li.classList.add('list-item');

// 追加
const ul = document.querySelector('ul');
ul.appendChild(li);                 // 末尾に追加（旧来）
ul.append(li, 'テキストも追加可');  // 末尾に追加（モダン）
ul.prepend(li);                    // 先頭に追加
ul.insertBefore(li, ul.firstChild); // 指定要素の前に挿入

// 削除
li.remove();                        // 要素自身を削除（モダン）
ul.removeChild(li);                // 子要素を削除（旧来）

// クローン
const clone = li.cloneNode(true);   // true で子要素も含めてコピー
ul.appendChild(clone);

// DocumentFragment（複数要素を一括追加してパフォーマンス向上）
const frag = document.createDocumentFragment();
for (let i = 0; i < 100; i++) {
  const item = document.createElement('li');
  item.textContent = `アイテム ${i}`;
  frag.appendChild(item);
}
ul.appendChild(frag); // 1回のDOM更新で100件追加
📌 試験頻出：append vs appendChild の違い appendChild(node)：ノードのみ受け付け、追加した要素を返す（旧来のAPI）
append(...nodes)：複数ノード・文字列も受け付け、返り値なし（モダンAPI）
prepend / before / after / replaceWith もモダンAPIとして覚えておきましょう。

innerHTML vs textContent — XSSリスクを理解する

innerHTML はHTML文字列として解釈するため、 ユーザー入力をそのまま代入するとXSS（クロスサイトスクリプティング）攻撃の原因になります。 ユーザー入力を表示する場合は必ず textContent を使いましょう。

プロパティ動作XSSリスク用途
innerHTMLHTML文字列として解釈・レンダリング⚠️ あり信頼できるHTMLの挿入
textContentテキストとして扱う（HTMLエスケープ）✅ なしユーザー入力の表示
innerText表示されているテキストを取得/設定✅ なしスタイルを考慮したテキスト
outerHTML要素自身を含むHTML文字列⚠️ あり要素ごと置換

dataset によるカスタムデータ属性

data-* 属性はHTML要素に任意のデータを埋め込む標準的な方法です。 JavaScriptからは element.dataset プロパティでアクセスできます。 属性名のハイフン（-）はキャメルケースに変換されます。 

<!-- HTML -->
<div id="product"
     data-product-id="42"
     data-price="1500"
     data-in-stock="true">
  商品名
</div>

// JavaScript からアクセス
const el = document.getElementById('product');

console.log(el.dataset.productId);  // → '42'（ハイフン→キャメルケース）
console.log(el.dataset.price);      // → '1500'（常に文字列）
console.log(el.dataset.inStock);    // → 'true'（文字列！boolean ではない）

// 書き込み・削除
el.dataset.userId = '999';          // data-user-id="999" が追加される
delete el.dataset.price;             // data-price 属性が削除される
⚠️ dataset の値は常に文字列 data-count="5" を dataset.count で取得しても文字列 "5" です。 数値として使う場合は Number(el.dataset.count) や parseInt() で変換が必要です。
